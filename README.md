# Vue Market

Commercial City Project Based on Vue.

<code>node v8.9.1</code> &nbsp; <code>yarn v1.7.0</code> &nbsp; <code>OS macOS</code>

#### Build Setup

``` bash
# install dependencies
$ yarn install

# serve with hot reload at 127.0.0.1:8081
$ yarn run dev

# run production
$ yarn run build
```
